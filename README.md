# Uocial

Social Media for University of Crete students.

![Demo](./assets/demo.gif)

## What is Uocial?

Uocial (UoC + Social), is a social media platform, specifically designed for the
needs of UoC students.

At the moment, it offers anonymous posts and comments (aka "ανομολόγητα"). The
following areas are planned for the near future:

- course reviews
- random, real-time, chat
- meet new people/students
- cultural groups (aka "ομάδες φοιτιτηκού κέντρου") info, anouncements etc
- student political group (aka "φοιτητικές παράταξεις") info, anouncements etc
